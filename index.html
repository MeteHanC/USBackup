USB Backup Script With Python

USBackup is simple backup program written with Python 3.4.Here how it works ; When you plug a removable drive in it detects it and copies all the files to C drive as new folders(Usbackup-1,Usbackup-2 ......)

Startup

Its also automaticaly adding the program to startup folder so the program will be executied when windows starts.

Converting to Exe with Using py2exe module

And you can convert it to exe without having any issues. So it will run on any computer.
